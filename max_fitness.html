Find point of maximum fitness within some range.

Usage

max_fitness(bounds, p, log_scale = TRUE, tol = 0.001)

Arguments

bounds
Two element vector specifing range within which to search
p
Parameters object to use. Importantly, the strategy_default element gets used here.
log_scale
Is the parameter naturally on a log scale? If so, this will greatly speed things up.
tol
Tolerance used in the optimisation

Description

Find point of maximum fitness in empty fitness landscape within a specified range.

Author

Daniel Falster, Rich FitzJohn
