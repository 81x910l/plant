Trait bounds

Usage

bounds(...)


check_bounds(bounds, finite = FALSE)


bounds_infinite(trait_names)


check_point(x, bounds)

Arguments

...
Named list, each element of which is a 2-element numeric vector of lower and upper bounds.
bounds
A set of bounds
finite
Logical indicating if bounds must be finite
trait_names
Character vector of trait names
x
a point to detect if it lies within bounds

Description

Helper function for making bounds

Examples

bounds(lma=c(0.01, 10))



    lower upper
lma  0.01    10



bounds(lma=c(0.01, 10), rho=c(1, 1000))



    lower upper
lma  0.01    10
rho  1.00  1000
